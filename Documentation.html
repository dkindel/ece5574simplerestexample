/api/robots/
  • POST
  • OPTIONS
  • None

Appends a new robot object to the list of robot objects. Returns the newly created robot object and its attributes. Attributes of the new robot object
are assigned default values and ids corresponding to its location in the robot list.

/api/robots/
  • OPTIONS
  • DELETE
  • None

Deletes all robot objects in the robot object list. Returns a Boolean indicating the success of the deletion

/api/robots/<int:r_id>/
  • HEAD
  • OPTIONS
  • GET
  • r_id

Returns the JSON robot object with the given r_id value. If no robot objects are being stored or no robot object
with the given r_id value could be located, a message is returned informing the user that no such robot object exists.

/api/robots/<int:r_id>/sensors/

  • OPTIONS
  • r_id

Appends a new sensor object to the list of sensor objects associated with the robot object with the given r_id. Returns the newly created sensor object and

Returns the sensor object with the given snsr_id associated with the robot object with the given r_id. The sensor is listed with its

/api/robots/<int:r_id>/sensors/<int:snsr_id>/
  • OPTIONS
  • DELETE
  • r_id
  • snsr_id

Deletes the sensor object from the robot object with the corresponding snsr_id and r_id, respectively. Returns the new list of sensor objects associated
with the robot object, confirming the deletion of the sensor object. If no robot objects are being stored or no objects with the given id values could be located,
a message is returned informing the user that no such robot or sensor object exists.

/api/robots/<int:r_id>/status/
  • HEAD
  • OPTIONS
  • GET
  • r_id

Returns just the status of the robot object with the given r_id. The status of a robot is a numerical value that
corresponds to a status code. These status codes can quickly identify how a robot is moving, if it has detected a threat,
as well as how the robot has responded to a threat. If no robot objects are being stored or no robot object with the given
r_id value could be located, a message is returned informing the user that no such robot object exists
